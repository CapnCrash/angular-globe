<m2s-globe
points="{Array}" watched - Array of point groups to be drawn. Point group is a object containing array property values, where each element is point object.
map-data="{Object}" watched - If you load the directive with angular-globe-with-maps.js you don't need to use this. Otherwise, if you want to provide custom "land", put here GeoJSON data.
map-type="{land|countries}" - When you loaded directive with angular-globe-with-maps.js you can specify, whether to show only land or country borders also. See Advanced Example
size="{Number}" - Specify size of a globe. Width equals to height.
point-stroke="{Expression} watched" - Define stroke color of a point. Expression has available point object as d.
point-stroke-width="{Expression} watched" - Define stroke width of a point. Expression has available point object as d.
point-fill="{Expression} watched" - Define fill color of a point. Expression has available point object as d.
point-class="{Expression} watched" - Define class of a point. Expression has available point object as d.
point-id="{Expression} watched" - Define id attribute of a point. Expression has available point object as d.
point-lat="{Expression} watched" - Define latitude of a point. Expression has available point object as d.
point-lng="{Expression} watched" - Define longitude of a point. Expression has available point object as d.
point-radius="{Expression} watched" - Define radius of a point. Expression has available point object as d.
graticule="{Boolean} default:true watched" - Show/Hide graticule
draggable="{Boolean} default:false watched" - If globe is draggable
sensitivity="{Number} default:0.25 watched" - Sensitivity of rotation while dragging
clickable="{Boolean} default:true watched" - If points are clickable
point-click="{Expression}" - Executes when point is clicked. Expression has available 3 variables:
d - Point object
c - Screen coordinates (not lat/lng) of click event, relative to globe svg element
el - D3 element of clicked point
raised-land="{Boolean} default:true" - If raised land effect is turned on
rotate="{Boolean} default:false watched" - If globe should rotate
rotate-speed="{Boolean} default:1 watched" - Globe rotation speed
rotate-angle="{Array} watched" - Will rotate globe once by given degrees. Array should contain exactly 2 elements (Rotation around x and y axis)
enter-anim-delay="{Expression}" - Delay of transition for newly added points to map. Expression has available 2 variables:
d - Point object
i - Index of point
Note, index point can be effectively used to create stagger transition. See Earthquakes Example
exit-anim-delay="{Expression}" - Delay of transition for removed points from the map. See enter-anim-duration above
enter-anim-duration="{Expression}" - Duration of transition for newly added points to map. Expression has available 2 variables:
d - Point object
i - Index of point
exit-anim-duration="{Expression}" - Duration of transition for removed points from the map
enter-anim-ease="{String}" - Ease of fadein transition for newly added points. See D3 easing
exit-anim-ease="{String}" - Ease of fadeout transition for removed points.
></m2s-globe>

Styling

These classes are automatically added to globe, so you can use it for CSS styling:
.m2s-globe - SVG element of a globe
.m2s-globe-water - Element representing water
.m2s-globe-graticule - Element representing graticule
.m2s-globe-shadow - Shadow of raised land
.m2s-globe-land-bottom - Bottom side of raised land
.m2s-globe-land-top - Top side of land
.m2s-globe-points - All points on globe have this class
